# Module: Analytics

## Overview
Provides dashboards, KPIs, and business intelligence for all ERP modules.

## Key Features
- Customizable dashboards
- KPI tracking
- Data visualization (charts, graphs)
- Cross-module analytics (HR, sales, inventory, etc.)
- Export and reporting tools
- Scheduled reports
