# Module: Studio

## Overview
Provides customization, low-code/no-code tools, and workflow automation for the ERP system.

## Key Features
- Custom form builder
- Workflow designer
- Dynamic field and model creation
- UI customization tools
- Automation rules
- Integration with other modules
